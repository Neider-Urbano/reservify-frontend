# Reservify Frontend

Este es el frontend del sistema de reservas Reservify, construido con Angular y Tailwind CSS.

## Tecnologías Utilizadas

- **Angular**
- **Tailwind CSS**
- **TypeScript**

## Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/reservify-frontend.git
   ```
